# CollegeDesk

Веб-приложение для управления учебным процессом в колледже, интегрированное с Telegram.

## Функциональность

- Расписание занятий
- Оценки
- Посещаемость (для преподавателей)
- Система жалоб и предложений
- Новости

## Технологии

- React
- TypeScript
- Telegram WebApp API
- Railway для деплоя

## Установка и запуск

1. Установка зависимостей:
```bash
npm install
```
